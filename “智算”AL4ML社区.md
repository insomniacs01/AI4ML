“智算”AI4ML社区
一、项目背景 
自动机器学习（AutoML）极大地加速了 AI 开发，但现有的 AutoML 系统通常配置繁琐，对缺乏编程和算法背景的领域专家（如医学、商科学生）门槛极高 。虽然目前有一些基于大语言模型（LLM）的辅助工具，但它们往往只能解决单一环节（如特征工程或超参数调优），且缺乏高效的规划能力 。
“智算”平台是一个基于多智能体协作（Multi-Agent）的 AI4ML 低代码/零代码开发社区。它参考了最前沿的 AutoML-Agent 框架 ，旨在让用户仅通过自然语言描述和上传原始数据，就能由背后的“智能体开发团队”自动完成数据预处理、模型搜索、训练以及部署的端到端工作 。平台不仅是一个开发工具，更是一个供师生交流、沉淀优质数据集和模型工作流的开源社区。
二、系统需求
1. 社区与系统管理员：
•	能够管理平台的用户权限与 API Token 额度，防止多智能体在自动化搜索和代码生成过程中消耗过多的计算资源。
•	能够管理社区的“数据中心”与“模型广场”，对用户公开分享的数据集和生成的 AI 预测模型进行审核、分类（如：图像分类、表格回归、时间序列预测等）及打标签。
2. 零基础的业务领域用户（如：想预测农作物产量的农学学生）：
•	能够通过简单的自然语言对话输入任务需求（例如：“请帮我用这份土壤数据集建立一个预测产量的模型，要求准确率尽量高”），并上传用户的 CSV 数据集。平台内的Agent 能自动解析用户的需求并结构化 。
•	在等待期间，能通过直观的可视化界面看到背后的“智能体团队”的工作进度。例如，看到 Agent Manager 正在进行任务拆解，Data Agent 正在分析数据并决定填补缺失值 ，以及 Model Agent 正在挑选最适合的模型架构（如 XGBoost 或 ResNet）。
•	最终能一键获得一个可以直接在网页上测试的部署接口（Web Demo），并且系统能输出一份通俗易懂的模型性能和特征重要性分析报告。
3. 有一定经验的 AI 开发者/数据科学家：
•	平台能够提供“人机协同（Human-in-the-loop）”的干预机制。在智能体团队执行“多阶段验证（Multi-Stage Verification）”时 ，用户可以在请求验证、执行验证或代码实现验证的环节介入 ，修改智能体规划的子任务，或者手动调整某些特征工程的逻辑。
•	能够查看和修改 Operation Agent 最终生成的完整 Python 源代码 ，以便用户将其下载到本地的高性能服务器上进行二次开发。
•	能够将用户手动优化过的整套工作流（Pipeline）和提示词模板分享到社区，供其他用户“Fork”和复用。
